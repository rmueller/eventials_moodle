### Plugin webinar Eventials (Aulas síncronas)

Esse plugin permite criar aulas síncronas no moodle integrado com a plataforma Eventials.

Basta criar uma conta na plataforma eventials.

Esse webinar tem as seguintes funcionalidades
 - Adicionar/editar/remover um webinar
 - Adicionar o professor como palestrante
 - Inscrever os alunos no webinar ao acessarem a atividade
